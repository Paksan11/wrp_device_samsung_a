# How-to build:
```
$ export ALLOW_MISSING_DEPENDENCIES=true
$ . build/envsetup.sh
$ lunch omni_a10e-eng
$ mka recoveryimage
```
## How to find the image built
```
`cd /out/target/product/a20`
```
see recovery.img
